# Register-Diagram

레지스터 비트 필드 정보를 입력하면 draw.io(.drawio) 다이어그램 파일을 생성하는 웹 도구입니다.

## 사용법

GitHub Pages: https://ye-eun-kang.github.io/Register-Diagram/

1. Register Name과 Address(선택)를 입력합니다.
2. Bit Range(예: `31:24`, `7`)와 Field Name을 입력합니다. Field Name을 비워두면 Reserved 처리됩니다.
3. **draw.io 파일 생성** 버튼을 누르면 `.drawio` 파일이 다운로드됩니다.

모든 처리는 브라우저 안에서만 이루어지며, 입력한 데이터는 서버로 전송되지 않습니다.
